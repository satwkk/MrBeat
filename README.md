# Mr Beat

A simple discord music bot with multi guild support.

## Commands available

Command Prefix = "-"
```
play <song name>
play <url> (Supported platforms are youtube, spotify for now !)
queue
list_queue (lq, l)
swap
flush
skip
users
totalmessages
leave
owner
```
